Proyecto 1: Modelo de Comestibilidad de Hongos

Este proyecto es parte de mi trabajo final para un curso de ciencia de datos, donde clasifiqué diferentes hongos según su comestibilidad.

Fuente del Conjunto de Datos

El conjunto de datos utilizado es el conjunto de datos de clasificación de hongos de Kaggle, que proporciona una descripción completa de las características de los hongos.

Técnicas de Preprocesamiento de Datos

  • Verificaciones de Valores Nulos: Se aseguró la integridad del conjunto de datos manejando adecuadamente los valores faltantes.
  • Codificación de Variables Categóricas: Transformé las características categóricas en valores numéricos para un entrenamiento efectivo del modelo.

Algoritmos de Aprendizaje Automático

  • Random Forest Classifier
  • Support Vector Machine (SVM)
  • Logistic Regression

Proyecto 3: Análisis de Obras Públicas en Excel

En este proyecto, realicé un extenso análisis de datos de obras públicas en Argentina, con el objetivo de obtener información sobre la distribución y el estado de las inversiones en infraestructura a nivel regional.

Fuente del Conjunto de Datos

El conjunto de datos fue obtenido del Mapa de Inversiones del Ministerio de Obras Públicas de Argentina, que proporciona información detallada sobre las obras públicas en diversas provincias y su estado de avance.

Técnicas de Análisis y Preprocesamiento de Datos

  • Limpieza y Validación de Datos: Se eliminaron errores y se validaron tanto los valores cualitativos como cuantitativos para garantizar la precisión del análisis.
  • Tablas Dinámicas: Se utilizaron para facilitar la exploración de datos y obtener respuestas a preguntas clave, como el número de obras por tipo, estado y ubicación geográfica.
  • Segmentación de Datos: Se añadieron segmentadores para permitir la segmentación dinámica de los datos por categorías como tipo de obra, año, estado de ejecución y provincia.

Visualización y Tablero

Se creó un tablero visual interactivo en Excel con gráficos y segmentadores que permiten una rápida visualización de la distribución y el estado de las obras públicas:

  • Distribución de Obras por Provincia: Destaca las provincias con la mayor inversión en obras.
  • Clasificación de Obras por Estado: Muestra el avance de las obras (planificadas, en progreso, completadas).
  • Análisis por Tipo de Obra: Identifica las categorías prioritarias de obras en la inversión pública.

Resultados Clave

Este análisis proporcionó una visión general del estado de las obras públicas en Argentina, destacando las regiones con mayor y menor inversión, así como los tipos de proyectos más comunes y su nivel de avance. Estas perspectivas son valiosas para entender las tendencias de inversión y las posibles áreas de mejora en la gestión de obras públicas.
