Project 1: Mushroom Edibility Model

Descripción:

Este proyecto es parte de mi proyecto final para un curso de ciencia de datos, en el cual clasifiqué diferentes hongos en base a su comestibilidad.

Dataset Source:

El dataset utilizado es el Mushroom Classification dataset de Kaggle, que provee una descripción detallada de las características de los hongos.

Técnicas de Preprocesamiento de Datos:

  • Verificación de valores nulos: Asegura la integridad del dataset al gestionar los valores faltantes adecuadamente.
  • Codificación de variables categóricas: Transformación de características categóricas en valores numéricos para el entrenamiento efectivo de modelos de machine learning.

Algoritmos de Machine Learning:

  • Random Forest Classifier
  • Support Vector Machine (SVM)
  • Logistic Regression

Project 2: Weather Prediction Model

Descripción:

En este proyecto, realicé un análisis exhaustivo de un dataset sintético diseñado para clasificar distintos tipos de clima.

Técnicas de Preprocesamiento de Datos:

  • Manejo y imputación de valores nulos: Mantenimiento de la calidad del dataset.
  • Codificación de variables categóricas: Conversión de variables categóricas en un formato adecuado para el entrenamiento de modelos.
  • Eliminación de outliers: Aplicación del método IQR para eliminar valores atípicos.

Algoritmos de Machine Learning:

  • Random Forest Classifier
  • Gradient Boosting Machine (GBM)
  • Logistic Regression
  • Decision Tree Classifier

Optimización y Evaluación de Modelos:

  • Precisión: Medición de la exactitud general de las predicciones.
  • Matriz de Confusión: Evaluación del rendimiento de los modelos de clasificación.

Se realizó una comparación de modelos para identificar el mejor ajuste para la predicción del tipo de clima, además de la optimización de hiperparámetros para mejorar los resultados.
